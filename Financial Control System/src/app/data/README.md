# Dados da empresa

Edite o arquivo **`empresa.ts`** nesta pasta para preencher os dados da sua empresa. Eles aparecem nos fechamentos impressos (cliente) e em outros comprovantes.

## Campos disponíveis

| Campo         | Uso                              |
|---------------|-----------------------------------|
| `nome`        | Razão social ou nome da empresa  |
| `nomeFantasia`| Nome fantasia                    |
| `cnpj`        | CNPJ (se for empresa)            |
| `cpf`         | CPF (se for MEI/autônomo)         |
| `ie`          | Inscrição estadual               |
| `endereco`    | Logradouro                       |
| `numero`      | Número                           |
| `complemento` | Complemento                      |
| `bairro`      | Bairro                           |
| `cidade`      | Cidade                           |
| `estado`      | UF (ex: SP)                      |
| `cep`         | CEP                              |
| `telefone`    | Telefone                         |
| `email`       | E-mail                           |
| `site`        | Site                             |

Deixe em branco (`""`) o que não for usar. O cabeçalho dos impressos mostra apenas o que estiver preenchido.
